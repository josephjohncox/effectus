# Security Policy

## Reporting a Vulnerability
If you believe you have found a security issue, please report it privately.

Preferred:
- Open a GitHub Security Advisory for this repository.

Alternative:
- If advisories are not available, contact the maintainers through a private channel (email or a private issue).

Please include:
- A clear description of the issue and potential impact.
- Steps to reproduce or a proof of concept.
- Affected versions or commit SHAs.

## Response Process
We will acknowledge reports as soon as possible and coordinate a fix or mitigation. Please avoid public disclosure until a fix is available.

## Supported Versions
Security updates apply to the current `main` branch. If you require backports, note that in your report.
